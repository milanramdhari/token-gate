# token-gate

Token-gate is an open-source AI API gateway and routing layer for developers and businesses building AI-powered applications.
It exposes a single unified endpoint, issues custom API keys, and verifies credits before forwarding requests to any upstream LLM provider.
Rate limiting is enforced by both credits and token usage, giving you fine-grained control over cost and abuse.
Built as a self-hostable alternative to OpenRouter — no 5% platform fee, no vendor lock-in, just your infrastructure.
Own your AI routing layer completely.
